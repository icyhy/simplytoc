## 项目概述

Vue Task Manager 是一个功能丰富的任务管理应用，旨在帮助个人和团队更好地组织和跟踪工作进度。该项目采用现代化的前端技术栈，提供直观的用户界面和强大的功能。

## 核心功能

### 任务管理
- **任务创建与编辑**: 支持富文本编辑器，可添加详细描述
- **优先级设置**: 高、中、低三级优先级标识
- **截止日期**: 智能提醒和逾期标识
- **任务状态**: 待办、进行中、已完成、已取消

### 项目组织
- **项目分组**: 按项目或类别组织任务
- **看板视图**: 类似 Trello 的拖拽式任务管理
- **列表视图**: 传统的任务列表展示
- **日历视图**: 基于时间的任务安排

### 团队协作
- **成员管理**: 邀请团队成员加入项目
- **任务分配**: 将任务分配给特定成员
- **评论系统**: 任务讨论和进度更新
- **通知中心**: 实时消息推送

## 技术特色

- **响应式设计**: 完美支持移动端和桌面端
- **实时同步**: WebSocket 实现多用户实时协作
- **离线支持**: PWA 技术支持离线使用
- **数据可视化**: 项目进度和统计图表
