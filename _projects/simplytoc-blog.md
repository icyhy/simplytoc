## 项目简介

SimplyToc Blog 是一个基于 Jekyll 的个人博客平台，使用了经典的 Hux 主题。该项目专注于提供优雅的阅读体验和强大的内容管理功能。

## 主要特性

- **响应式设计**: 完美适配桌面端和移动端设备
- **标签云功能**: 智能的文章标签分类和展示
- **社交媒体集成**: 支持多种社交平台的分享和链接
- **自动化部署**: 通过 GitHub Pages 实现自动构建和部署
- **SEO 优化**: 内置搜索引擎优化功能

## 技术栈

- **Jekyll**: 静态站点生成器
- **Liquid**: 模板引擎
- **Bootstrap**: 前端 UI 框架
- **LESS**: CSS 预处理器
- **GitHub Pages**: 托管平台

## 项目亮点

1. **优雅的设计**: 采用简洁现代的设计风格，注重用户体验
2. **高性能**: 静态站点确保快速的加载速度
3. **易于维护**: 清晰的代码结构和完善的文档
4. **可扩展性**: 模块化的设计便于功能扩展

这个项目展示了如何使用 Jekyll 构建一个功能完整、设计精美的个人博客系统。
