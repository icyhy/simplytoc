# 说在前面的话
今天是圣诞平安夜，也是pmp培训课程最后一天。正式的上课就结束了，剩下还有3次模拟考试，几次小组活动，然后就是明年3月底的最终大考。今晚决定放纵一晚，不碰教材，不看讲义，总之和pmp有关的书一概不看，写完这篇博客就去看看电影，打打游戏，争取早点上床好好补个觉。明天就是新的开始。

# 体会
连续两个周末，第一个周末2天，这个周末从周五开始到今天，一共5天的课程，早上9到下午4点半，每天其实大约5个半小时，上午下午课间休息15分钟，中午休息1个半小时。   

唯一的感觉就是累。头两天上课，感觉脑袋还比较清醒，一方面开头几章比较容易理解，同时书看的也比较多。但到了这周，连续3天课程，真感觉有点吃不消。教材虽然提前通读了一遍，但是如走马观灯般的阅读，脑子里留下的实在不多，上课的进度飞快，稍不留神，可能几个知识点就过去了，需要笔记的地方很多，几乎不停的在记，从第一天上课到现在，两根签字笔油用完了。有图有真相，这是今天那根。
![输入图片说明]({{ site.baseurl }}/img/in-post/2017-12-23-pmp2-214602_b9860ce6_58701.jpeg "1.jpg")   
总之，脑子大部分时间都处于紧张状态，手也没怎么停，这样的强度，对于我现在来说有点吃力了。不过好在坚持下来了。内心里，对我自己听课的质量是能给一个及格分的，毕竟没有虚度光阴，尽最大可能去听了。   

这一段时间每天晚上看书的时间控制在1-1.5小时，加上上班地铁上看书的时间，感觉这个时间投入应该足够了，毕竟PMP只是我年度计划的一部分，没必要把其他所有计划都停了。   

上课虽然累，但是感觉还是挺有意思的。第一，离开学校多年，很久没有这么正经的上课备考了，依稀找回点学校的感觉（只是依稀），只要摸摸我的草包肚子，这种感觉顿时消失的无影无踪。第二，老师确实给力。陈钢老师1400场的培训经验果然不是浪得虚名。感觉PMP的理论已经烂熟于胸，随口说来的话都严丝合缝，让人感觉到体系严谨的“美”。另外老师书本外的实践体会也讲的很透彻，很多地方能让人会心一笑，不是理论加实践很多年的人，应该说不出这样的话来。老师秀他自己的加油记录、装修记录等等“清单”，也非常真实。确实是把项目管理的很多理念身体力行了，讲的很多就是他自己的心里话，不是为了培训拼凑素材，这一点尤其让人反思，习惯的养成其实对人自身的提升价值更大，远比考过一个证，读过几本书有价值，老师的一些日常习惯（如拉清单、做计划、沟通、谈判、寻求专家帮助的解决问题思路等）为他工作成功及生活幸福发挥了很大的作用。至于是先有的习惯，后来学习了理论去巩固习惯，还是先学了理论，之后逐步培养了习惯就不得而知了。我要不要尝试着培养一些好的生活习惯，是我接下来要尝试甚至于教导我的孩子去尝试的的事情。   

今天培训结业了，大家拍照留念，也拿到了培训完成证书。秀一下，其实时间还是太短，同一个组的同学们还大都叫不上名字，昨天小组活动去吃饭，也就身边的几个人记住名字，多聊了几句。希望多年后，看到照片，还能记着这次培训过程中见过的人，去过的地方，做过的事。

# 下一步计划
好吧，上课体会就这么多，毕竟备考的路才开了个头，离走完还早呢，接下到1月14日第一次模拟考试还有3周的时间。计划把教材和讲义全部精读一边，尤其是要将讲义上的知识点和书上的对照着看看先。这个阶段还是看书为主，“强书”里面带着的习题做做就行了，不打算大批量的做题。到下一个阶段再考虑增加题量。
