# 前因
不知道从什么时候开始，我的字就被定位成狗爬了，我也接受了这个定位。当然，有时候说我的字像狗爬都是夸奖我。从小学到高中，写字就是图快，始终也没有耐性去练字。唉，现在知道，胡写其实并不快，要想真正的快，必须遵守书写的规律和规则，那是前人总结出来的经验。可惜等我知道这个道理，已经错过了最好的练字时机。大学乃至工作后，用键盘的时间远超过用笔的时间，书写速度和水平逐年下降，这几年已经到了提笔忘字的阶段了，有时候不得不写字的时候，甚至发现手不受控制的抖动，笔画也完全不听使唤。这时候看到身边写字漂亮的同事或朋友，心里是十分羡慕的。

于是有一天，终于做了个决定，从今年起，我要开始练字。先练钢笔字，目标不要太高，先练到书写工整，脱离狗爬的行列吧。我知道“字无百日功”，我也做好了一直练下去的准备。

# 经过
暂时没有报班，找老师的打算。也不太关注网上争论的各种大师的字体的好坏。只觉得练字（硬笔）就要用钢笔。先从楷书开始，练一阵，再开始练行书。再过一阵，回过头练楷书，交替着练。先从钢笔字帖练起，等入门了，有了一定的基础了，再照着古人的毛笔字帖练。从下决心，买字帖，付诸行动开始，到今天接近100天了，就算百日吧。在此做一个阶段性小结。另外，虽然过了百日，其实字还根本不能见人，练字的状态也完全无法融入到平时的书写中，导致平时一写字，就回到过去的不受控状态，估计是病入骨髓，康复远非一朝一夕。因此，从现在开始，每隔一段时间，记录一下自己的状态，心得，也秀两张图片，自我勉励督促一下，看看随着时间的流逝，字体是否有所改进，想要放弃的时候，也再看看此文，想想自己的决定。也许多年后，回头看看，也会觉得有意思呢。

# 体会
- 姿势很重要，从握笔到坐姿。刚开始的时候只注意握笔姿势，没注意坐姿，后来发现坐姿不好，容易疲倦。而且如果胸口压着桌边的话，影响呼吸。
- 呼吸很重要，刚开始写字，有时候会憋着气写，过于紧张和用力。一度觉得是不是笔出问题了，怎么笔画这么粗。有一阵写的食指天天疼。尤其是憋气，容易紧张，猛地换气的时候，字一定出状况。后来意识到，写字时候，呼吸要平和，心情要平和，不急，也不能太慢。
- 读帖很重要，我现在的问题就是读帖。大概一看觉得差不多就跳过去写下一字了，或者总感觉字不像，但就是不知道问题出在哪里了。道理貌似知道，每天写字不用求多，要仔细读帖，一点一点纠正，看来我还没有到那个状态。
